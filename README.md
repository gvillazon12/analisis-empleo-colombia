# 📊 Análisis de empleo formal e informal en Colombia

## 📌 Descripción del proyecto
Este proyecto tiene como objetivo analizar las características socioeconómicas de los jefes de hogar trabajadores domésticos en Colombia para el año 2023, con énfasis en la diferenciación entre empleo formal e informal.

Se realiza un enfoque estadístico integral que combina análisis descriptivo, técnicas multivariadas y modelos econométricos para identificar patrones relevantes y factores asociados al tipo de empleo.

---

## 🎯 Objetivo
Identificar y analizar los determinantes del empleo formal e informal en Colombia, considerando variables como edad, sexo, nivel educativo, ingresos, tipo de contrato y zona geográfica.

## 📊 Metodología

El análisis se desarrolló en las siguientes etapas:

### 1. Limpieza y preparación de datos
- Tratamiento de valores faltantes  
- Transformación de variables  
- Creación de variables categóricas  

### 2. Análisis descriptivo
- Distribución de jefes de hogar por departamento  
- Caracterización por edad, sexo, estado civil y nivel educativo  
- Análisis de ingresos y experiencia laboral  

### 3. Visualización de datos
Se generaron gráficos para facilitar la interpretación de los resultados, incluyendo:
- Distribución del tipo de empleo  
- Análisis por sexo y grupos de edad  
- Clasificación por niveles de ingreso  

### 4. Análisis multivariado
- Análisis de Correspondencias Múltiples (MCA) para identificar relaciones entre variables categóricas  

### 5. Modelos estadísticos
Se implementaron diferentes modelos para profundizar el análisis:

- **Modelo Probit:** para estimar la probabilidad de pertenecer a empleo formal o informal  
- **Modelos de supervivencia:**
  - Kaplan-Meier  
  - Modelo de Cox  
- **Descomposición Oaxaca:** para analizar diferencias en ingresos  

---

## 📈 Resultados principales
El análisis evidencia que variables como el nivel educativo, el tipo de contrato, la edad y la zona de residencia tienen una influencia significativa en la probabilidad de pertenecer al empleo formal o informal.

Asimismo, se identifican diferencias estructurales en ingresos y condiciones laborales entre ambos grupos.
